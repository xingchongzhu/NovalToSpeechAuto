# 小说有声剧本格式参考文档

***

## 一、文档说明

本文档为《剑来》有声小说剧本格式的**标准化、可自动化生成**规范，所有后续章节剧本必须严格遵循。

本规范明确**人声与音效边界**、**分段规则**、**音色一致性**、**BGM/音效用法**，确保合成稳定、无报错、可直接批量生产。

**参考模板文件**：`小说批量工具/剧本生成skill/剧本模版.json`

***

## 二、顶层结构

| 字段                 | 类型     | 必填 | 说明                 |
| ------------------ | ------ | -- | ------------------ |
| `project`          | string | 是  | 项目名称，如"剑来有声小说自动生成" |
| `chapter`          | string | 是  | 章节名称，如"第1章 惊蛰"     |
| `global`           | object | 是  | 全局配置               |
| `roles_definition` | object | 是  | 角色定义（全局统一）         |
| `data`             | array  | 是  | 剧本片段数组（按顺序播放）      |

***

## 三、全局配置 (`global`)

```json
{
  "pitch": "+0Hz",
  "channels": 1,
  "voice_volume": "+0%",
  "bgm_volume": "-18%",
  "effect_volume": "-10%"
}
```

| 字段              | 说明                 | 默认值    |
| --------------- | ------------------ | ------ |
| `pitch`         | 全局音调偏移             | `+0Hz` |
| `channels`      | 音频通道数（1=单声道，2=立体声） | `1`    |
| `voice_volume`  | 人声基准音量             | `+0%`  |
| `bgm_volume`    | 背景音乐基准音量           | `-18%` |
| `effect_volume` | 音效基准音量             | `-10%` |

***

## 四、角色定义 (`roles_definition`)

### 核心规则

1. **全局一致性**：同一个角色在整部小说中，`role_voice`/`pitch`/`volume`/`speed`必须完全一致。
2. **角色定义优先级**：优先使用《剑来角色配配音表.md》中的角色定义保证前后角色统一；如果找不到定义则使用《克隆音频角色列表说明.md》配音进行配音定义。
3. **音色来源**：所有`role_voice`必须来自《克隆音频角色列表说明.md》，禁止自定义音色。
4. **人声发声原则**：**笑声、哭声、叹气、惊呼、喘息、冷哼**等拟人发声，**全部由人声演绎，不做音效**。

| 角色   | 角色描述      | 音色          | 语速   | 音调   | 音量  |
| ---- | --------- | ----------- | ---- | ---- | --- |
| 旁白   | 沉稳大气、古风叙事 | 云健-中年男性磁性声音 | -10% | 0Hz  | 0%  |
| 陈平安  | 少年主角、温和坚韧 | 云-男青年多语言    | -2%  | 0Hz  | 0%  |
| 宋集薪  | 傲气少年、玩世不恭 | 云扬-男青年      | +2%  | +3Hz | +2% |
| 稚圭   | 怯弱少女、声音轻柔 | 云夏-女青年      | -1%  | +4Hz | -1% |
| 锦衣少年 | 外乡少年、从容随性 | 云杰-男青年      | 0%   | +1Hz | +1% |
| 吴爷爷  | 随行老者、沉稳和蔼 | 云泽-男中年      | -8%  | -6Hz | 0%  |

## 五、剧本片段 (`data`)

每个片段固定结构：

```json
{
  "id": 序号,
  "role": "角色名",
  "api": {
    "voice": {},
    "bgm": {},
    "effects": []
  },
  "mix": { "mode": "" }
}
```

***

## 六、语音配置 (`voice`)

```json
{
  "text": "台词原文",
  "role": "角色名",
  "role_voice": "音色名",
  "speed": "",
  "volume": "",
  "pitch": "",
  "instruct": "语气指导"
}
```

### 人声发声禁区（严禁做音效）

以下所有拟人发声**必须由人声演绎，禁止添加音效**：

- 笑声（哈哈大笑、轻笑、嗤笑、坏笑）
- 哭声、抽泣、哽咽
- 叹气、冷哼、嗤鼻
- 惊呼、倒吸冷气、喘息
- 语气拟声词（哇、啊、哎、哼、呵）

***

## 七、背景音乐配置 (`bgm`)

```json
{
  "play_mode": "keep/switch/lower/fade",
  "scene": "场景标识",
  "scene_cn": "中文描述",
  "scene_en": "英文描述",
  "fade_in": 0-2,
  "fade_out": 0-2,
  "volume": "-18%",
  "pitch": "+0Hz"
}
```

### 播放模式说明

| 模式       | 说明        | 适用场景    |
| -------- | --------- | ------- |
| `keep`   | 继续播放当前BGM | 场景延续    |
| `switch` | 切换到新BGM   | 场景转换    |
| `lower`  | BGM降低音量   | 对话场景    |
| `fade`   | BGM淡出     | 场景结束、过渡 |

BGM常见规则

&#x20;BGM，不压人声

&#x20;

- BGM / 音效切换必分段
- 场景 / 时间变化必分段
- 情绪转折必分段
- 插入音效必分段
- 拟人发声不做音效，由人声完成
- 旁白 → 平静叙述
- 冲突 / 调侃 → 情绪张力
- 诗句 / 结尾 → 情感升华
- 同一段情绪不切换 BGM

### 常用BGM参考场景

| 场景标识 | 中文描述   | 氛围特点    |
| ---- | ------ | ------- |
| 小镇暮色 | 古风小镇夜晚 | 轻柔古筝与竹笛 |
| 静夜星空 | 古风静夜   | 轻柔古琴    |
| 悬念氛围 | 神秘古风   | 低沉古琴    |
| 深夜独处 | 静夜古琴   | 孤独氛围    |
| 希望微光 | 悠远希望   | 温暖治愈    |
| 街巷对峙 | 轻微紧张   | 张力感     |

## 八、音效配置 (`effects`)

### 允许使用的音效（仅非人声、环境/动作类）

烛火摇曳、桃枝敲打、吹灭蜡烛、关门声、脚步声、跳下墙头、抛掷钱袋、窑炉熄火、劈竹声、水流声、风吹声、鸟声、风声、雨声等。。。

### 禁止使用的音效

任何人声/拟人发声：笑声、哭声、叹气、冷哼、喘息、嘶吼

### 音效字段规范

```json
{
  "trigger_delay": 数字,
  "duration": 数字,
  "process_mode": "insert",
  "name": "动作+材质",
  "sound_cn": "中文描述",
  "sound_en": "英文描述",
  "volume": "-20%",
  "pitch": "+0Hz"
}
```

### 触发延迟计算原则

- 旁白语速 -10% ≈ 3字/秒
- 延迟时间（秒）= 文字对应动作位置字数 ÷ 3

### 音效命名规范

**格式**：动作 +（材质/力度）

**示例**：跳下墙头、木门关闭、脚尖点地、钱袋抛掷

***

## 九、混音模式 (`mix`)

| 模式             | 说明          | 适用场景           |
| -------------- | ----------- | -------------- |
| `mix`          | 人声+BGM+音效   | 正常场景           |
| `voice_on_bgm` | 人声+BGM，关闭音效 | 对话、纯叙述场景       |
| `voice_only`   | 仅人声         | 开场标题、结尾独白、重点台词 |
| `effect_only`  | 仅音效         | 转场、空镜过渡        |

***

## 十、自动化分段强制规则

1. 角色切换必分段
2. BGM切换/模式变更必分段
3. 场景/时间/地点变化必分段
4. 情绪明显转折必分段
5. 插入动作音效必分段
6. 单句文本＞40字建议拆分
7. 拟人发声（笑/叹/哭）不分音效，直接写进`text`与`instruct`

***

## 十一、创作与命名规范

1. **文件名格式**：`剑来第X章-章节名.json`
2. **角色名**：使用全称，不缩写、不简写、无错别字
3. **文本要求**：100%还原原著，不增删、不改写、不添加注释
4. **音效命名**：简洁规范，动作+材质
5. **全局一致**：同一角色音色、语速、音调、音量全程不变

***

## 十二、自动化生成校验清单（生成后必查）

- 所有`role_voice`来自官方克隆音色表
- 无任何人声类音效（笑/哭/叹/哼/喘息）
- BGM使用规范场景标识
- 分段符合角色切换/BGM切换规则
- `id`从0开始连续、不重复、不跳号
- 音量遵循：人声 > 音效 > BGM
- 文本100%还原原著无修改
