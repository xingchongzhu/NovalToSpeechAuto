# 小说批量工具

## 项目需求

小说批量工具是一个用于自动批量合成小说语音的工具，旨在为小说内容添加背景音和音效，使生成的音频更加生动有趣，提升听众的听觉体验。

### 核心需求
1. **自动语音合成**：将小说文本转换为自然流畅的语音
2. **背景音添加**：为不同场景添加合适的背景音乐
3. **音效增强**：根据小说内容智能添加音效
4. **批量处理**：支持处理多个小说章节
5. **质量控制**：确保生成的音频质量清晰、自然

## 功能介绍

小说批量工具支持自动批量合成小说语音，为小说内容添加背景音和音效，使生成的音频更加贴合小说内容和场景。

### 主要功能

1. **全流程自动化**：实现小说文本解析→章节剧本生成→角色语音分配→场景化语音合成→背景音/音效匹配→音频合成→文件结构化存储全流程自动化
2. **智能角色语音分配**：基于文本内容自动识别角色并分配对应语音
3. **场景化语音合成**：根据场景类型适配语速和声调
4. **批量语音合成**：支持批量处理小说剧本文件，自动生成语音
5. **智能背景音生成**：根据场景类型自动生成或选择合适的背景音
6. **智能音效推荐**：根据小说内容分析推荐合适的音效
7. **音频混合**：将语音、背景音和音效智能混合
8. **章节级配置**：支持为每个章节设置不同的音频配置
9. **场景分析**：智能分析小说场景，推荐合适的音频配置
10. **结构化存储**：自动按章节组织音频文件和临时文件

## 使用方法

### 1. 准备小说剧本

按照小说剧本格式创建或修改小说剧本文件，确保包含必要的配置信息。

### 2. 运行脚本

```bash
# 进入脚本目录
cd "小说批量工具/脚本"

# 赋予脚本执行权限
chmod +x "小说语音自动批量合成工具.sh"

# 运行脚本处理指定的小说剧本
./小说语音自动批量合成工具.sh "../小说剧本/剑来/剑来第一章-入山门青峰山.json"

# 或运行脚本处理所有小说剧本
./小说语音自动批量合成工具.sh
```

### 3. 查看生成结果

脚本运行完成后，生成的音频文件会保存在 `audio/小说名称/合成音频/` 目录中。

## 脚本使用说明

### 脚本功能

小说语音自动批量合成工具.sh 是一个用于自动批量合成小说语音的脚本，主要功能包括：

1. **批量处理**：支持处理单个或多个小说剧本文件
2. **智能分析**：分析小说内容，推荐合适的背景音和音效
3. **音频生成**：生成或下载背景音和音效
4. **音频混合**：将语音、背景音和音效混合
5. **结果输出**：将合成后的音频保存到指定目录

### 运行参数

- **无参数**：处理小说剧本目录下所有小说的JSON文件
- **JSON文件路径**：处理指定的小说剧本文件

### 运行示例

```bash
# 处理所有小说剧本
./小说语音自动批量合成工具.sh

# 处理指定小说剧本
./小说语音自动批量合成工具.sh "../小说剧本/剑来/剑来第一章-入山门青峰山.json"
```

## 注意事项

1. **ffmpeg依赖**：脚本需要ffmpeg来生成和混合音频，请确保已正确安装
2. **目录结构**：请按照指定的目录结构组织文件，否则脚本可能无法正常运行
3. **脚本权限**：运行前请确保脚本文件有执行权限
4. **资源消耗**：处理大量段落时可能会消耗较多系统资源，请确保系统资源充足
5. **配置文件**：请确保小说剧本文件格式正确，否则可能导致脚本运行失败

## 故障排除

### 常见问题

1. **脚本无法运行**
   - 检查脚本权限是否正确
   - 检查ffmpeg是否已安装
   - 检查目录结构是否正确

2. **音频生成失败**
   - 检查小说剧本格式是否正确
   - 检查系统资源是否充足
   - 检查网络连接是否正常（如果需要下载资源）

3. **音频质量问题**
   - 调整背景音和音效的音量
   - 选择更合适的背景音类型
   - 检查小说剧本中的语音配置
