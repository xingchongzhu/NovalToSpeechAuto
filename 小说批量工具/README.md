# 剑来有声小说自动生成系统

## 项目概述

本项目是一个基于Qwen3-TTS和EasyVoice的有声小说自动生成系统，专门用于将小说文本转换为高质量的有声读物。系统支持多种语音角色、背景音和音效的混音，能够生成具有专业水准的有声小说。

## 项目结构

```
小说批量工具/
├── 小说剧本/            # 小说剧本JSON文件目录
│   └── 剑来/           # 剑来小说剧本
├── 脚本/               # 核心脚本目录
│   ├── audio_processing_module.py    # 音频处理模块
│   ├── novel_batch_executor.py       # 批量执行脚本
│   └── novel_batch_manager.sh        # 批处理管理脚本
├── output/             # 输出目录（自动生成）
└── README.md           # 项目说明文档
```

## 核心功能

- **语音合成**：支持Qwen3-TTS和EasyVoice两种引擎
- **多角色配音**：支持多种角色声音，包括不同年龄、性别和地域特色
- **背景音和音效**：支持添加背景音乐和环境音效
- **智能混音**：支持多种混音模式，实现语音、背景音和音效的完美融合
- **批量处理**：支持批量处理多个章节的有声小说生成

### 配置参数说明

- **project**: 项目名称
- **chapter**: 章节名称
- **global**: 全局参数
  - **pitch**: 音调调整
  - **channels**: 声道数
  - **voice_volume**: 语音音量
  - **bgm_volume**: 背景音音量
  - **effect_volume**: 音效音量
- **roles_definition**: 角色定义
  - **desc**: 角色描述
  - **fixed_voice**: EasyVoice引擎参数
  - **qwen_voice**: Qwen3-TTS引擎参数
- **data**: 台词数据
  - **id**: 台词ID
  - **role**: 角色名称
  - **api**: API参数
    - **voice**: 语音参数
      - **params**: EasyVoice参数
      - **qwen_params**: Qwen3-TTS参数
    - **bgm**: 背景音参数（可选）
    - **effects**: 音效参数（可选）
  - **mix**: 混音配置

## 使用方法

### 1. 准备小说剧本

在`小说剧本`目录下创建对应小说的子目录，然后创建章节JSON文件，按照上述JSON结构填写内容。

### 2. 安装依赖

```bash
# 安装Python依赖
pip install pydub ffmpeg-python requests

# 安装FFmpeg（用于音频处理）
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt-get install ffmpeg

# Windows
# 下载FFmpeg并添加到环境变量
```

### 3. 运行批处理脚本

```bash
# 进入脚本目录
cd 小说批量工具/脚本

# 运行批处理脚本
./novel_batch_manager.sh
```

### 4. 查看输出结果

生成的有声小说将保存在`output`目录下，按照小说名称和章节名称组织。

## 技术实现

### 音频处理模块

`audio_processing_module.py`是系统的核心模块，负责：

- 初始化TTS引擎（Qwen3-TTS或EasyVoice）
- 文本转语音
- 音频参数调整（语速、音量、音调）
- 音频混音
- 背景音和音效处理

### 批处理执行

`novel_batch_executor.py`负责批量处理多个章节的有声小说生成，支持并行处理以提高效率。

### 混音模式

系统支持多种混音模式：

- **voice_only**: 只有语音，无背景音和音效
- **bgm_fade_in_then_voice**: BGM淡入后播放语音
- **voice_on_bgm**: 语音叠加在BGM上
- **mix**: 语音、BGM和音效混合

## 常见问题与解决方案

### 1. Qwen3-TTS模型初始化失败

**解决方案**：
- 确保模型路径正确
- 检查CUDA环境是否配置正确
- 如果仍然失败，系统会自动切换到EasyVoice引擎

### 2. 音频生成失败

**解决方案**：
- 检查网络连接（EasyVoice需要网络连接）
- 检查FFmpeg是否正确安装
- 检查临时目录权限

### 3. 角色声音不符合预期

**解决方案**：
- 在JSON配置中调整角色的语音参数
- 尝试使用不同的角色声音
- 调整语速、音量和音调参数

### 4. 背景音和音效问题

**解决方案**：
- 调整背景音和音效的音量
- 调整混音模式
- 检查音效文件是否存在

## 项目扩展

### 添加新角色

1. 在`qwen3-tts-base-model/clone-audio`目录下添加新的角色音频文件
2. 在小说剧本JSON文件中添加角色定义
3. 在台词数据中使用新角色

### 自定义背景音和音效

1. 在台词JSON文件的`bgm`和`effects`部分添加自定义内容
2. 确保场景描述清晰，以便系统生成合适的背景音
