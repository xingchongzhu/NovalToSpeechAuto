# 小说语音自动批量合成工具使用说明

## 工具介绍

小说语音自动批量合成工具是一个用于自动批量合成小说语音的脚本工具，它可以将小说文本转换为语音，并为其添加背景音和音效，使生成的音频更加生动有趣。

## 功能特点

1. **批量处理**：支持处理单个或多个小说剧本文件
2. **智能分析**：分析小说内容，推荐合适的背景音和音效
3. **音频生成**：生成或使用已有的背景音和音效
4. **音频混合**：将语音、背景音和音效智能混合
5. **结果输出**：将合成后的音频保存到指定目录

## 系统要求

- **操作系统**：支持Linux、macOS
- **依赖软件**：ffmpeg（用于音频处理）
- **网络连接**：如需下载资源，需要网络连接

## 安装准备

### 1. 安装ffmpeg

**macOS**：

```bash
brew install ffmpeg
```

**Linux**：

```bash
# Ubuntu/Debian
sudo apt-get install ffmpeg

# CentOS/RHEL
sudo yum install ffmpeg
```

### 2. 准备目录结构

确保项目目录结构如下：

```
小说批量工具/
├── audio/                  # 音频文件目录
│   └── 剑来/               # 小说名称目录
│       ├── bgm/            # 背景音目录
│       ├── sfx/            # 音效目录
│       ├── 原始音频/         # 原始音频文件
│       └── 合成音频/         # 合成后的音频文件
├── 小说剧本/                # 小说剧本目录
│   └── 剑来/               # 小说名称目录
│       ├── 剑来第一章-入山门青峰山.json  # 小说剧本文件
│       └── 剑来第二章-修行开始.json  # 小说剧本文件
├── 脚本/                   # 脚本目录
│   └── 小说语音自动批量合成工具.sh  # 主脚本文件
└── README.md              # 项目说明文档
```

> 注：临时文件目录会在运行时自动生成在 `audio/小说名称/章节名称/` 目录下

### 3. 设置脚本权限

```bash
chmod +x "小说批量工具/脚本/小说语音自动批量合成工具.sh"
```

## 使用方法

### 方法一：处理所有小说剧本

```bash
# 进入脚本目录
cd "小说批量工具/脚本"

# 运行脚本
./小说语音自动批量合成工具.sh
```

### 方法二：处理指定小说剧本

```bash
# 进入脚本目录
cd "小说批量工具/脚本"

# 运行脚本处理指定文件
./小说语音自动批量合成工具.sh "../小说剧本/剑来/剑来第一章-入山门青峰山.json"
```

## 输出结果

脚本运行完成后，生成的文件将保存在以下位置：

- **合成音频**：`小说批量工具/audio/小说名称/合成音频/`
- **背景音**：`小说批量工具/audio/小说名称/bgm/`
- **音效**：`小说批量工具/audio/小说名称/sfx/`
- **原始音频**：`小说批量工具/audio/小说名称/原始音频/`
- **临时文件**：`小说批量工具/audio/小说名称/章节名称/temp_audio_segments/`（运行过程中生成，按章节组织）

## 常见问题

### 1. 脚本无法运行

**解决方案**：

- 检查脚本权限是否正确设置
- 检查ffmpeg是否已安装
- 检查目录结构是否正确

### 2. 音频生成失败

**解决方案**：

- 检查小说剧本格式是否正确
- 检查系统资源是否充足
- 检查网络连接是否正常（如果需要下载资源）

### 3. 音频质量问题

**解决方案**：

- 调整背景音和音效的音量
- 选择更合适的背景音类型
- 检查小说剧本中的语音配置

### 4. 脚本运行缓慢

**解决方案**：

- 减少同时处理的段落数量
- 确保系统资源充足
- 预生成常用的背景音和音效

## 故障排除

### 查看详细日志

脚本默认会输出详细日志，您可以通过日志了解脚本的运行状态和可能的错误原因。

### 检查临时文件

脚本会在运行过程中生成临时文件，您可以检查这些文件来了解处理过程：

- 临时音频片段：`temp_audio_segments/` 目录

### 验证目录权限

确保脚本有足够的权限读取和写入文件：

```bash
# 检查目录权限
ls -la "小说批量工具/audio"
ls -la "小说批量工具/小说剧本"

# 确保目录可写
chmod -R 755 "小说批量工具/audio"
chmod -R 755 "小说批量工具/temp_audio_segments"
```

## 优化建议

1. **预生成资源**：对于常用的背景音和音效，可以提前生成，减少运行时间
2. **合理配置**：根据小说内容合理配置背景音和音效，避免过度使用
3. **定期清理**：定期清理临时文件和不需要的音频文件，节省存储空间
4. **分批处理**：对于大型小说，可以分批处理，避免系统资源不足
